# India GDP Sectoral Forecasting

Forecasting GDP growth at sectoral level for Agriculture and Services
for short term (1Y), medium term (3-5Y) and long run (10Y).

## Data Sources
- MOSPI National Accounts Statistics
- RBI DBIE
- Haver Analytics
- World Bank

## Sectors
- Agriculture
- Services

## Models
- SARIMA
- VAR / ARDL
- XGBoost (scenario analysis)
